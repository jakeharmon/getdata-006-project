This codebook explains the following variables and how they are applied to the tidy data set created using the run_analysis.R script from this repository.

For more description on how the data was collected see features_info.txt from the Samsung dataset

"activity" =   one of the following: WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
